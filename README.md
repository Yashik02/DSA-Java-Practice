## 🛠️ Repository Roadmap

### 🏛️ Advanced Data Structures
* **Graphs:** Comprehensive library including **Dijkstra’s**, **Bellman-Ford**, **Prim’s**, **Kruskal’s**, **Kosaraju’s**, and **Tarjan’s** algorithms. Supports Adjacency Lists and Cycle Detection.
* **Trees:** Implementations of **Binary Trees** (Diameter, LCA, Ancestors), **Tries** (Prefix Trees for word-break & unique substrings), and **Segment Trees** (Range Sum/Min queries).
* **Linear Structures:** Custom **Doubly Linked List** with cycle removal and **Stacks/Queues** featuring sliding window optimizations.
* **System Design:** Functional **LRU Cache** using a hybrid HashMap and Doubly Linked List approach.

### ⚙️ Algorithm Design
* **Dynamic Programming:** Tabulation and Memoization for **0-1 Knapsack**, **Longest Common Subsequence**, **Matrix Chain Multiplication**, and **Catlan Numbers**.
* **Recursion & Backtracking:** Deep dives into **N-Queens**, **Sudoku Solver**, **Rat in a Maze**, and the **Knight’s Tour**.
* **Greedy Algorithms:** Optimization problems like **Fractional Knapsack**, **Activity Selection**, and **Chcola** cost minimization.
* **Sorting & Searching:** 10+ algorithms including **QuickSort**, **MergeSort**, **HeapSort**, and **Counting Sort**, plus Rotated Binary Search.

### ☕ Core Java & OOPs
* **OOP Principles:** Detailed implementations of **Inheritance**, **Encapsulation**, **Abstraction**, and **Polymorphism**.
* **Advanced Logic:** **Bit Manipulation** (Ith bit operations, Power of 2), String Compression, and Anagram checkers.
